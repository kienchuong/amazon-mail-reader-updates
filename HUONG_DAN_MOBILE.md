# Mobile Dashboard

Mobile Dashboard la trang web rieng cho dien thoai, khong dang nhap Gmail, Outlook hoac Yahoo.

## Cai dat Apps Script

1. Mo Google Sheet Payment Royalties > **Tien ich** > **Apps Script**.
2. Mo file `google_sheets_webhook.gs` di kem ban app 0.5.0, xoa code cu va dan toan bo code moi.
3. Giu nguyen gia tri `SECRET` dang dung trong code Apps Script. Secret nay phai trung voi Secret o tab **Cai dat** cua app PC.
4. Bam **Trien khai** > **Quan ly cac tu chon trien khai** > chon ban dang hoat dong > bam bieu tuong sua.
5. Chon **Phien ban moi**, sau do bam **Trien khai**. Khong tao URL moi; URL `/exec` cu van dung.

## Bat Mobile Dashboard tren PC

1. Cap nhat Amazon Mail Reader len 0.5.0.
2. Mo **Cai dat** > **Mobile Dashboard**.
3. Nhap PIN it nhat 4 ky tu, bam **Dat PIN**.
4. Bam **Dong bo ngay** sau khi da quet mail. Tu lua chon **Tu dong dong bo sau khi quet** da bat san.
5. Bam **Mo dashboard**, sau do mo cung URL nay tren dien thoai.

## Them shortcut

- Android Chrome: mo menu 3 cham > **Them vao man hinh chinh**.
- iPhone Safari: bam **Chia se** > **Them vao Man hinh chinh**.
- Mac Safari/Chrome: mo URL va dung **Add to Dock** hoac **Create shortcut**.

Lan dau tren moi thiet bi, nhap PIN. Lan sau shortcut se mo truc tiep.

## An toan

- App dien thoai chi doc snapshot do PC dong bo; khong co thong tin dang nhap email.
- Bam **Thu hoi thiet bi** tren PC khi doi may, mat dien thoai hoac muon buoc nhap PIN lai.
- Neu doi `SECRET`, cap nhat cung gia tri do o app PC truoc khi dong bo tiep.
